# Andew-Project
- 6명이서 만든 의류 쇼핑몰 입니다.
# 사용한 기술
- Frontend
  - HTML/css
  - Javascript
  - Bootstrap
- Backend
  - Java
  - Spring
  - Jquery/Javascript/Ajax
- Database
  - MySQL
  - MyBatis
- Server
  - Apache Tomcat
<br/><br/><br/>
# 나의 구현 기능
- 로그인 기능
- 회원 가입
- 회원 탈퇴
- 회원정보 수정
- 회원 목록(관리자용)
  - 수정
  - 삭제
- 나의 활동내역
- 운영 관리
